# Fast Food Nutrition Data Analysis

This project analyzes fast food nutrition data using Python and Data Science techniques.

## Dataset
The dataset is taken from Kaggle and contains nutritional information like calories, fat, sugar, sodium, and protein.

## Techniques Used
- Data Cleaning
- Data Transformation
- Exploratory Data Analysis (EDA)
- Data Visualization

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab
